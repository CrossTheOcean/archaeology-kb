## 二、数据层设计

### 2.1 数据存储方案

采用**混合存储架构**：结构化数据存关系数据库/图数据库，非结构化文本存向量数据库。

| 数据类型 | 存储方案 | 用途 |
|----------|----------|------|
| 实体元数据 | SQLite / PostgreSQL | 精确查询、统计聚合 |
| 关系三元组 | Neo4j (图数据库) | 关系 traversal、路径查询 |
| 实体上下文 | FAISS (向量索引) | 语义相似度检索 |
| 原始PDF | 文件系统 / 对象存储 | 溯源、原文展示 |
| 标注报告 | Markdown / JSON | 人可读、可导出 |

### 2.2 Neo4j 图模型设计

```cypher
// 节点类型
(:Site {name: "曾侯乙墓", province: "湖北", era: "东周", entity_count: 10222})
(:Unit {name: "M1", type: "墓葬", page: 15})
(:Artifact {name: "鼎", category: "铜器", count: 243})
(:Measurement {value: "21.3米", dimension: "长度"})
(:Era {name: "战国", period: "东周"})
(:Province {name: "湖北", region: "华中"})

// 关系类型
(:Site)-[:CONTAINS]->(:Unit)
(:Unit)-[:EXCAVATES {page: 15}]->(:Artifact)
(:Site)-[:DATES_TO]->(:Era)
(:Unit)-[:MEASURES]->(:Measurement)
(:Site)-[:LOCATED_IN]->(:Province)
(:Artifact)-[:MADE_OF]->(:Material)
```

## 三、检索策略详解

### 3.1 问题分类与路由

系统首先需要识别用户问题的类型，路由到不同的检索策略：

| 问题类型 | 示例 | 检索策略 |
|----------|------|----------|
| **事实查询** | "曾侯乙墓出土了多少件铜器？" | 结构化SQL查询实体统计 |
| **列表查询** | "哪些遗址出土了编钟？" | 图数据库关系遍历 |
| **比较查询** | "曾侯乙墓和妇好墓哪个铜器更多？" | 多源数据聚合+对比生成 |
| **定义查询** | "什么是层位关系？" | 向量语义检索+上下文生成 |
| **推理查询** | "为什么东周时期铜器数量激增？" | 多文档聚合+LLM推理 |
| **地理查询** | "河南省有哪些新石器时代遗址？" | 结构化过滤+地理排序 |

### 3.2 多路召回策略

```python
def multi_way_retrieval(query: str, top_k: int = 10):
    """多路召回：同时执行多种检索，结果融合"""
    
    # 路1: 实体精确匹配
    exact_results = entity_exact_match(query)
    
    # 路2: 关系三元组查询
    graph_results = graph_traversal(query)
    
    # 路3: 向量语义检索
    vector_results = vector_similarity_search(query, top_k=20)
    
    # 路4: 关键词全文检索
    keyword_results = full_text_search(query)
    
    # 融合排序 (Reciprocal Rank Fusion)
    fused_results = reciprocal_rank_fusion([
        exact_results, graph_results, 
        vector_results, keyword_results
    ])
    
    return fused_results[:top_k]
```

### 3.3 检索示例

**用户问题**: "哪些遗址出土了编钟？编钟最多的遗址是哪个？"

**检索过程**:
1. **意图识别**: 列表查询 + 聚合统计
2. **实体链接**: "编钟" → 关联到铜器类别中的"钟"
3. **图查询** (Neo4j Cypher):
   ```cypher
   MATCH (s:Site)-[:CONTAINS]->(u:Unit)-[:EXCAVATES]->(a:Artifact {name: "钟"})
   RETURN s.name, count(a) as bell_count
   ORDER BY bell_count DESC
   ```
4. **向量检索**: 搜索上下文含"编钟""钟磬""礼乐"的文本片段
5. **结果生成**:
   > 根据71本考古报告标注数据，以下遗址出土了编钟：
   > 
   > 1. **曾侯乙墓**（湖北随州）：出土钟1,556件，为71本报告中编钟数量最多的遗址。其中包含著名的曾侯乙编钟，共64件，分三层八组悬挂。
   > 2. **张家坡西周墓地**（陕西西安）：出土钟...件
   > 3. ...
   > 
   > 编钟最多的遗址是**曾侯乙墓**，其编钟数量占71本报告所有"钟"类实体的绝大部分，反映了战国时期曾国高度发达的礼乐制度。
   > 
   > *数据来源：《曾侯乙墓》考古报告，p.4-p.500*

## 四、生成层设计

### 4.1 系统Prompt设计

```python
ARCHAEOLOGY_SYSTEM_PROMPT = """你是一位资深的考古学专家，精通中国新石器时代至宋元明清的考古学知识。
你的回答基于《考古学专刊·丁种》71本报告的结构化标注数据（共324,735个实体、443,060条关系）。

回答规则：
1. 所有事实性陈述必须标注数据来源（报告名称+页码）
2. 数字必须精确，不得编造
3. 对于不确定的信息，明确说明"基于现有数据无法确定"
4. 使用专业但易懂的语言
5. 适当引用考古学概念时，给出简要解释

可用数据类型：
- 遗址信息：名称、省份、年代、经纬度、总页数、实体数
- 遗迹信息：编号、类型（墓葬/房址/灰坑等）、层位关系
- 器物信息：类型（陶器/铜器/铁器等）、数量、材质、纹饰
- 测量数据：长度、面积、角度等
- 年代信息：绝对年代、相对年代、文化类型

当前日期：2026年4月
"""
```

### 4.2 答案生成流程

```
用户问题
   │
   ▼
检索Top-K相关实体/关系/上下文片段
   │
   ▼
构建Prompt = 系统提示 + 检索结果 + 用户问题
   │
   ▼
LLM生成答案
   │
   ▼
后处理：添加引用标注、格式化输出
   │
   ▼
返回给用户
```

### 4.3 答案质量保障

| 保障机制 | 方法 | 效果 |
|----------|------|------|
| **事实核查** | 生成后二次检索验证关键数字 | 减少幻觉 |
| **引用溯源** | 每个数据点后附 [报告名, p.页码] | 可追溯 |
| **置信度标注** | 对不确定信息标注"推测"/"可能" | 诚实性 |
| **拒绝机制** | 对超范围问题（如未收录遗址）明确拒绝 | 可靠性 |

## 五、典型问答示例

### Q1: 事实查询
**问**: "蒙城尉迟寺遗址有多少个遗迹编号？"

**检索**: SQL查询 `entity_stats.遗迹编号`

**答**: "蒙城尉迟寺遗址共有**3,624个**遗迹编号（《蒙城尉迟寺》考古报告，p.1-p.489）。这些遗迹编号涵盖房址、灰坑、墓葬、基址等多种类型，反映了该聚落遗址复杂的空间布局。"

---

### Q2: 关系查询
**问**: "M54墓葬出土了哪些器物？"

**检索**: 图查询 `MATCH (u:Unit {name: "M54"})-[:EXCAVATES]->(a:Artifact) RETURN a`

**答**: "根据《安阳殷墟花园庄东地商代墓葬》报告，M54出土的器物包括：铜器289件（含鼎、爵、觚等礼器）、陶器...、玉器...（p.XX-p.XX）。M54为商代中型墓葬，器物组合反映了..."

---

### Q3: 比较查询
**问**: "曾侯乙墓和张家坡西周墓地，哪个铜器更多？"

**检索**: 分别查询两墓的铜器数量

**答**: "**曾侯乙墓**的铜器数量远多于张家坡西周墓地：
- 曾侯乙墓：4,582件铜器（《曾侯乙墓》，p.4-p.530）
- 张家坡西周墓地：1,884件铜器（《张家坡西周墓地》，p.4-p.712）

曾侯乙墓铜器数量是张家坡的2.4倍。这一差异主要反映了：
1. 时代差异：东周时期青铜铸造技术已高度成熟
2. 等级差异：曾侯乙为诸侯级别，随葬规格更高
3. 功能差异：曾侯乙墓以礼乐器（编钟1,556件）为主，张家坡以礼容器为主"

---

### Q4: 推理查询
**问**: "为什么东周时期铜器数量比西周多？"

**检索**: 聚合多报告的年代-器物数据 + 相关上下文

**答**: "基于71本考古报告的标注数据，东周时期铜器数量显著多于西周，主要原因包括：

**数据支撑**：
- 西周（张家坡）：1,884件铜器
- 东周（曾侯乙墓）：4,582件铜器，单墓即超过西周墓地总量

**考古学解释**：
1. **技术进步**：东周时期失蜡法普及，可以铸造更复杂的器物（如曾侯乙编钟）
2. **礼制变化**：西周'列鼎制度'严格，东周'礼崩乐坏'，诸侯僭越使用高规格礼器
3. **经济繁荣**：铁器开始普及，青铜从工具领域解放，更多用于礼器和奢侈品
4. **区域发展**：楚国（曾侯乙为曾国，附属于楚）的青铜铸造业特别发达

*注意：71本报告中西周样本较少（仅4本），东周样本较多（13本），样本量差异也可能影响统计结果。*"
